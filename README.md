# mica mqtt 组件
基于 `t-io` 实现的 `mqtt` iot组件。  

## 使用
目前仅仅是试验性质，不过 `t-io` 确实很稳，`mica-mqtt-example` 中有 mqtt 服务端和客户端测试代码，main 方法运行即可。
